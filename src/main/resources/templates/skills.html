Слой баз данных приложения

PostgreSQL

Свободная объектно-реляционная система управления базами данных. Это на данный момент единственная СУБД, с которой мне приходилось работать.


Интересный опыт

  • Работа со специфичными для этой СУБД типами (inet, enums, arrays)
  • Написание запросов в процедурном стиле с помощью языкового расширения PL/pgSQL

Hibernate

Реализация JPA спецификации, предназначенная для решения задач объектно-реляционного отображения. Из всех известных мне фреймворков, думаю, Hibernate я знаю лучше всего.


Интересный опыт

  • Оптимизация запросов
  • Реализация адаптеров (AttributeConverter/UserType) для работы со специфичными для PostgreSQL типами
  • Решение проблем с ленивой инициализацией путем расширения байткода (bytecode enhancement)

Hibernate Envers

Модуль Hibernate, позволяющий отслеживать изменение сущностей. Нужен для осуществления контроля над такими критическими элементами как платежи и права, а также регистрировать действия пользователей/администраторов.


Интересный опыт

  • Расширение сущности ревизий для хранения большего количества служебной информации
  • Настройка аудита согласно условиям (conditional auditing)

Hibernate Search

Модуль Hibernate для осуществления полнотекстового поиска (надстройка над Apache Lucene).


Интересный опыт

  • Настройка индексирования сущностей по условию
  • Настройка анализатора и запроса с нечетким приближением поиска

Flyway

Библиотека, обеспечивающая версионность и управление миграциями баз данных. Нужно для синхронизации состояния БД у всей команды разработчиков.

Сервисный слой приложения

Java

Мой основной язык программирования. Использую Java 8.


Интересный опыт

  • Разработка и отладка многопоточного проекта

Spring Framework

Универсальный фреймворк, обеспечивающий IoC и DI и многие другие дополнительные возможности, такие как управление транзакциями и упрощение тестирования.

Spring Data JPA

Модуль Spring, обеспечивающий расширенные возможности взаимодействия с ORM-фреймворком, таким как Hibernate.

Spring Security

Модуль Spring, предоставляющий механизмы построения систем аутентификации и авторизации, а также другие возможности обеспечения безопасности для промышленных приложений.

Spring Boot

Модуль Spring для быстрой настройки и развертывания приложений. Также предоставляет ряд решений для повышения производительности разработки, таких как LiveReload и Hot Restart.

Spring MVC

Модуль для поддержки архитектуры модель-представление-контроллер (model-view-controller). Обеспечивает готовые компоненты, которые могут быть использованы для разработки веб-приложений.

Технологии генерации представлений

Thymeleaf

Современный серверный механизм Java-шаблонов. Отлично взаимодействует со Spring технологиями.

JSP

Технология, позволяющая веб-разработчикам создавать содержимое, которое имеет как статические, так и динамические компоненты.

Apache Wicket

Веб-фреймворк использующий подход, основанный на использовании компонентов. Пожалуй, это фреймворк, в применении которого я имею больше всего опыта.


Интересный опыт

  • Вызов изменения модели из JavaScript кода

Zk Framework

Фреймворк для разработки веб-приложений в событийно-ориентированной парадигме и компонентной модели. Позволяет писать интерфейсы в стиле MVVM, что делает разработку очень быстрой. Но при этом, из-за внутреннего устройства фреймворк не обеспечивает достаточной "мощности" для публичных сайтов. Поэтому во время работы этот фреймворк у нас использовался в основном для админок или корпоративных сайтов.

Вспомогательные серверные технологии

SLF4J

Библиотека для протоколирования, ставящая своей целью предоставить максимально простой, но при этом мощный фасад для различных систем протоколирования на Java

Logback

Фреймворк логирования, реализующий фасад SLF4j

Apache Velocity

Процессор шаблонов, предоставляющий простой, но одновременно гибкий шаблонный язык. В основном использовался мной для генерации e-mail сообщений

Jackson

Java библиотека для работы с данными в формате JSON

JAXB

Java библиотека для работы с данными в формате XML

Тестирование

JUnit

Библиотека для модульного тестирования программного обеспечения. Я работал как с 4, так и с 5 версиями этого фреймворка.

Mockito

Фреймворк, упрощающий создание заглушек компонентов системы с заранее установленным поведением.

Powermock

Фреймворк, предоставляющий дополнительные возможности для создание заглушек по сравнению с Mockito (№ задавать поведение приватных или же final методов)

Selenium

Библиотека, предоставляющая возможности тестирования end-to-end тестирования веб-интерфейса.

Фронтенд

Javascript

Язык программирования, с помощью которого страницы сайтов становятся интерактивными.

HTML 5

Блокчейн

Ethereum

Блокчейн-платформа для создания децентрализованных онлайн-сервисов на базе умных контрактов. Для тестирования использовался тестовый блокчейн Rinkeby.


Интересный опыт

Solidity

Объектно-ориентированный язык программирования самовыполняющихся контрактов для платформы Ethereum. De facto является стандартом для написания смартконтрактов.


Интересный опыт

  • Написание смартконтракта обменника эфира на токены (и наоборот) при привязке цены к внешнему фактору (рублю)
  • Отладка смартконтрактов с помощью Solidity IDE Remix
  • Использование Oraclize

Web3j

Библиотека, предоставляющая Java API к ethereum ноде.


Интересный опыт

  • Развернута тестовая среда TestRPC и тестовый приватный блокчейн, а также разработаны тесты для проверки проекта с повышенными требованиями к корректности данных
  • Разработан оракул для информирования смартконтракта об изменении внешнего параметра
